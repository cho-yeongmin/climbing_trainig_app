# 클라이밍 트레이닝 앱

Figma 선택 프레임 연동을 위한 반응형 웹 UI입니다.

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 으로 접속하세요.

## 반응형 브레이크포인트

- **모바일**: 기본 (320px~)
- **480px~**: 로고 텍스트 표시, 대시보드 2열
- **640px~**: 패딩·타이포 확대, Figma 미리보기 높이 증가
- **768px~**: 대시보드 3열
- **1024px~**: 여백·간격 확대

## Figma 연동

1. Figma 데스크톱 앱에서 원하는 프레임을 선택합니다.
2. Cursor 채팅에서 "선택한 디자인으로 코드 만들어줘" 또는 "선택한 프레임 스크린샷 보여줘"라고 요청합니다.
3. MCP가 현재 선택된 노드를 사용해 디자인 컨텍스트 또는 스크린샷을 반환합니다.
